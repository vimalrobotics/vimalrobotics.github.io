## Overview
A wearable IoT system designed for elderly safety and monitoring.
The system detects falls, monitors vitals, and provides emergency alerts.

## Features
- Heart rate monitoring
- GPS tracking and geofencing
- Fall detection
- Telegram-based alert system

## Key Learnings
- Reliability is critical in health-related systems
- Edge processing reduces response latency
