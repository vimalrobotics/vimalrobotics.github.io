## Overview
A bidirectional system enabling remote sharing of physiological signals and haptic feedback.
Designed to explore emotional communication over the internet.

## System Components
- Heart rate sensing
- Internet-based data transmission
- Haptic feedback devices
- Web application interface

## Learning Outcome
