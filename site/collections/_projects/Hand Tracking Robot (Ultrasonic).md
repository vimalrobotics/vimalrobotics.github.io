## Overview
A simple mobile robot capable of tracking hand motion using ultrasonic distance sensing.
The system supported both manual remote control and autonomous tracking modes.

## System Features
- Ultrasonic distance sensing
- Mode switching (manual / autonomous)
- Real-time motor control

## Learning Outcome
- Reactive control systems
- Sensor noise handling
- State-based mode management
