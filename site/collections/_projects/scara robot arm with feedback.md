## Overview
A SCARA robot controlled using camera-based feedback.
AprilTags were used for pose estimation and closed-loop inverse kinematics control.

## Technical Stack
- ROS
- AprilTag detection
- Inverse kinematics
- Servo-based actuation

## Key Contributions
- Vision-based feedback loop
- Calibration-aware control
- Real-time correction

## Learning Outcome
- Visual servoing complexity
- Importance of calibration and timing
