## Overview
A competition project developed for the Techgium challenge.
The system tracked and stabilized a crane hook using wireless sensing and a flight controller.

## System Components
- Flight controller
- Wireless ultrasonic sensors
- Ground station visualization

## Challenges
- Communication latency
- Sensor noise
- Real-time stabilization

## Outcome
- Reached final round of Techgium competition

## Key Learnings
- Real-world robotics demands robustness over elegance
