## Overview
An ESP32-CAM based system designed to detect human presence and provide audio-based greetings.
The system also guided users through calming exercises upon entry.

## Core Technologies
- ESP32-CAM
- Face detection algorithms
- Text-to-speech synthesis

## Applications
- Offices
- Educational institutions
- Public entry spaces

## Key Learnings
- Embedded vision constraints
- Human-centered system design
